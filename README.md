Requirements
------------
* node.js > 0.12
* Python >= 3.5

Development
-----------
1. Clone repo
2. `cd` in to repo
3. Run `./scripts/install_os_dependancies.sh`
4. Run `make venv`
5. Run `. venv/bin/activate`
6. Run `make`
7. Run `make start`
8. Go to `http://localhost:8000`
